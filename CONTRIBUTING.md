# Contributing

Contributions are welcome. To contribute:

1. Fork the repo and clone it locally.
2. Install dependencies: `npm install`
3. Run the app: `npm run dev`
4. Make your changes.
5. Run lint and typecheck:
   - `npm run lint`
   - `npm run typecheck`
6. Ensure `npm run build` succeeds.
7. Open a pull request. Your PR should pass CI (lint and typecheck).

Please keep the scope of PRs focused. For larger changes, open an issue first to discuss.
